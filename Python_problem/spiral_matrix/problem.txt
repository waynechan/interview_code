Given an integer n, return a matrix of n x n elements (n rows, n columns) in spiral order.

Example 1:

Input:3
Output:
[
 [ 1, 2, 3 ],
 [ 8, 9, 4 ],
 [ 7, 6, 5 ]
]

Example 2:

Input:4
Output:
[
 [ 1, 2, 3, 4 ],
 [ 12, 13, 14, 5 ],
 [ 11, 16, 15, 6 ],
 [ 10, 9, 8, 7 ]
]

Runtime: O(n^2)
Space: O(1)
